# Keyboard Key Detector

A simple JavaScript project that displays information about the key pressed by the user.

## Overview

Whenever a key is pressed, the application displays:

- **Key**
- **KeyCode**
- **Code**

The **Space** key is displayed as `"Space"` for better readability.

## Concepts Practiced

- `keydown` event
- Event object (`e`)
- DOM Manipulation
- Template Literals
- Dynamic HTML Updates

## How to Run

1. Open `index.html` in your browser.
2. Press any key on your keyboard.
3. View the key information displayed on the page.

## Learning Goal

This project was built to practice handling keyboard events and accessing properties of JavaScript event objects.
